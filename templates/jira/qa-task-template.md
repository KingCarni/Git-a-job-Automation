# Jira QA Task Template

## Title
QA test coverage for [feature]

## Type
Task

## Feature Area
Frontend / API / Auth / Data / Credits / AI / Parser

## Goal
Validate that [feature] works as expected and does not regress critical flows.

## Scope

### In Scope
- Functional testing
- Negative testing
- Regression testing
- Accessibility smoke checks
- API checks where applicable

### Out of Scope
- Production payment testing
- Exact AI wording validation
- Pixel-perfect visual testing unless specified

## Acceptance Criteria
- [ ] Happy path tested
- [ ] Negative cases tested
- [ ] Regression risks tested
- [ ] Auth behavior tested
- [ ] Data integrity checked
- [ ] Bugs filed with clear repro steps
- [ ] QA signoff added to PR/ticket

## Test Cases
Link or paste relevant test cases.

## Automation Candidates
List tests that should become Playwright/API tests.
